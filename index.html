Html
Тег Описание
a ссылка на сайта, используются атрибуты:
href="" ссылка открывается в новой владке
target="" ссылка открывается в новой владке
h1-h6 заголовки (чем больше значение, тем меньше шрифт)
strong полужирный шрифт
em курсив
small уменьшение шрифта
DOCTYPE html указывается версия HTML
html берет воедино всю страницу
head метаинформация
body тело страницы, все будет отображаться на странице
title используется совмество с head, дает название сайта название сайта
meta ввод кодировки, используются атрибуты:
сharset="UTF-8"
p создает параграф
br перенос на новую строку, ставиться в конце строки кода
ol упорядоченный список
ul неупорядоченный список
il пишется между тегами ol и ul
img вставка изображения, используются атрибуты:
src="" указывается источник файла или сайт
widht="***px" ширина изображения
height="***px" высота изображения
div разделение на блоки
table создание таблицы, используются атрибуты:
border="*" толщина обводки таблицы, указывается число
th название колон делается полужирным и посередине
tr указание строк
td указание колон
input создает поле для ввода, используются атрибуты:
value="..." стандартное значение
placeholder="..." прозрачный текст, например "Что ищем?"
textarea поле для ввода большого текста
audio создание аудиоплеера, используется атрибут:
control появляются кнопки управления
source источник, пишется в теге audio, используется атрибут:
src="..." вписывается источник аудиофайла
type="..." тип аудиофайла, например "audio/mpeg"
video аналогичен тегу audio, только создает видеоплеер
Css
Свойство Описание
color изменяет цвет текста
font-size изменение размера шрифта
background-color Изменение заднего фона
