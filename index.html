Шпаргалка HTML и CSS

HTML

HTML имеет теги и атрибуты.

Тег a - ссылка на сайта, используются атрибуты:
href="" - ссылка на сайт
target="" - ссылка открывается в новой владке

Тег h1-h6 - заголовки (чем больше значение, тем меньше шрифт)
Тег strong - полужирный шрифт
Тег em - курсив
Тег small - уменьшение шрифта
Тег DOCTYPE html - указывается версия HTML (В данном случае HTML 5)
Тег html - берет воедино всю страницу
Тег head - метаинформация
Тег body - тело страницы, все будет отображаться на странице
Тег title - используется совмество с head, дает название сайта название сайта
Тег meta - ввод кодировки, используются атрибуты:
сharset="UTF-8" -

Тег p - создает параграф
Тег br - перенос на новую строку, ставиться в конце строки кода

Тег ol - упорядоченный список
Тег ul - неупорядоченный список
Тег il - пишется между тегами ol и ul

Тег img - вставка изображения, используются атрибуты:
src="" - указывается источник файла или сайт
widht="***px" - ширина изображения
height="***px" - высота изображения

Тег div - разделение на блоки

Тег table - создание таблицы, используются атрибуты:
border="*" - толщина обводки таблицы, указывается число в
Тег th - название колон делается полужирным и посередине
Тег tr - указание строк
Тег td - указание колон

Тег input - создает поле для ввода, используются атрибуты:
value="..." - стандартное значение
placeholder="..." - прозрачный текст, например "Что ищем?"
Тег textarea - поле для ввода большого текста

Тег audio - создание аудиоплеера, используется атрибут:
control - появляются кнопки управления
Тег source - источник, пишется в теге audio, используется атрибут:
src="..." - вписывается источник аудиофайла
type="..." - тип аудиофайла, например "audio/mpeg"
Тег video - аналогичен тегу audio, только создает видеоплеер

CSS

CSS имеет селекторы и свойства.

Пишется в формате: "селектор {свойство: значение; свойство1: значение1; и т.д.}"

Свойства текста:

Свойство color - изменяет цвет текста
Свойство font-size - изменение размера шрифта
